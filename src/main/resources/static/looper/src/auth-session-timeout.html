Session Timeout

Session timeout and keep-alive control with a nice Bootstrap warning dialog.

The warning dialog will fire after 10 seconds. The page will redirect after 30 seconds with no activity.

Shows the warning dialog with countdown timer after 10 seconds. If user takes no action (interacts with the page in any way), browser is redirected to redirUrl. On any user action (mouse, keyboard or touch) the timeout timer as well as the countdown timer are reset (visible if you don't close the warning dialog).
